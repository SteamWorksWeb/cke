## IMsanity

## FREE!

![](/images/blog/imsanity2-1024x456.jpg)

Anyone who has ever run a WordPress website knows that media management can be a real headache. Imsanity is a free plugin that helps to automate the process of resizing images, making it a crucial tool for any site owner.

Perhaps one of the most notable features of Imsanity is its ability to resize images in bulk. For anyone who has ever had to individually resize a large number of images, this can be a real time-saver.

In addition, Imsanity also offers the option to convert images to the more web-friendly JPEG format and to delete the original, full-sized images after they have been resized. This helps to save storage space on your server, which can be valuable if you are on a limited budget.

Finally, Imsanity also has the ability to set a maximum width and height for all images uploaded to your site. This ensures that no excessively large images are ever uploaded, which can help to speed up your website’s load time.

Overall, Imsanity is an essential plugin for any WordPress website.

[GET IMSANITY NOW](https://wordpress.org/plugins/imsanity/)

## DUPLICATE POST

## FREE!

![](/images/blog/Duplicate.jpg)

If you’ve ever needed to duplicate a page or post on your WordPress site, you know how tedious it can be to copy and paste all the content by hand.

Fortunately, there’s a handy little plugin called Duplicate Page that makes the process a snap.

Just install the plugin and activate it, then go to the page or post you want to duplicate. Click on the “Duplicate” link in the WordPress admin bar, and voila! You’ll have an exact copy of the original page or post, complete with all its content, settings, and comments.

You can even choose to duplicate only certain parts of a page or post, which can come in handy if you need to create a similar but slightly different version of something.

So next time you need to duplicate a page or post on your WordPress site, be sure to give Duplicate Page a try – it just might save you hours of manual work.

[GET DUPLACTE PAGE NOW](https://wordpress.org/plugins/duplicate-page/)
